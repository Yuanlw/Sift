# 用户故事

## 写作者

作为写作者，我想把每天看到的文章、截图和观点保存到一个地方，这样我写作时可以快速找到相关材料和选题。

验收标准：

- 我可以从手机一键保存内容。
- 系统能自动提炼观点和例子。
- 系统能把内容关联到已有主题。
- 我可以问：“最近关于 AI Agent 的材料里有哪些可写选题？”

## 创业者

作为创业者，我想把市场信息、竞品动态、用户反馈和商业观点沉淀下来，这样我做判断时不会只依赖记忆。

验收标准：

- 我可以保存竞品网页和截图。
- 系统能归类到市场、竞品、用户需求、商业模式等页面。
- 系统能标记新资料和旧判断的冲突。
- 我可以问：“过去一个月我收集到的市场信号说明了什么？”

## 研究者

作为研究者，我想把论文、文章和笔记整理成可追溯的知识库，这样我写综述或做项目时能复用已有阅读。

验收标准：

- 每个知识点都能追溯到来源。
- 系统能区分事实、作者观点和我的解释。
- 系统能维护开放问题。
- 我可以问：“这个主题下还有哪些没有解决的问题？”

## 投资人或分析师

作为投资人或分析师，我想把公司、行业、访谈和新闻材料沉淀成结构化资料，这样我能更快形成判断。

验收标准：

- 系统能按公司、行业、人物和事件组织资料。
- 系统能保留来源日期。
- 系统能提醒同一主题下的冲突信息。
- 我可以问：“这家公司最近的积极信号和风险信号分别是什么？”
